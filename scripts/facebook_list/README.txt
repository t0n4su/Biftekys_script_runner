📌 Facebook Leads Script (list_script.py)

Αυτό το script επεξεργάζεται αρχεία Leads από τη Facebook και δημιουργεί αρχείο Excel για upload.

ΧΡΗΣΗ:
1. Κάντε drag & drop το αρχείο CSV της Facebook στη ζώνη εισαγωγής.
2. Επιλέξτε φάκελο εξόδου.
3. Πατήστε "Run Script".

ΑΠΟΤΕΛΕΣΜΑΤΑ:
✅ Δημιουργείται αρχείο Excel με όνομα `List_Upload_Vod.xlsx` με καθαρισμένα leads.

ΣΗΜΕΙΩΣΕΙΣ:
- Το script αφαιρεί αυτόματα το +30 από τα τηλέφωνα.
- Δεν απαιτείται επιπλέον ρύθμιση.
